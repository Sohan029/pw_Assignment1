Formatting Tags in HTML:

  1. Emphasis tag:
    • The HTML tag is used to emphasize the particular text in a paragraph.
    • Similar to italic, use to indicate emphasis or to differentiate between words or phrases
  2. Bold tag:
    • The tag is used to make the text bold.
    • Do not use for entire paragraphs or long blocks of text.
  3. Italic Tag:
    • The tag is used to make the text italic.
    • Avoid overusing, as it can become distracting or lose its impact.
  4. Underline Tag.:
    • The tag is used to set the text underline.
    • Avoid using for entire blocks of text, as it can make it difficult to read
